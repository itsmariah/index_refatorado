Recomendação de Setup Gamer

Esta página apresenta o meu projeto de setup gamer ideal. O objetivo é montar um computador de alto desempenho para jogar, fazer lives e trabalhar com programação.

Componentes do Computador

  • Processador: Intel Core i7 13700K
  • Placa de vídeo: RTX 5070
  • Memória RAM: 128GB DDR5 (distribuídos em 4 slots de 32GB)
  • Armazenamento: SSD NVMe 2TB
  • Fonte: 2000W 80 Plus Platinum
  • Gabinete: Hyte y70

Periféricos

  1. Monitor Principal 27" 240hz
  2. Monitor Lateral 27" 144hz
  3. Teclado mecânico RGB (HyperX, Logitech, Attack Shark)
  4. Mouse gamer sem fio (HyperX, Logitech, Attack Shark, Redragon)
  5. Headset com som surround
  6. Microfone para streaming
  7. Webcam 1080p
  8. Mousepad grande com iluminação RGB
  9. Soundbar com som surround
  10. Controles para jogos
